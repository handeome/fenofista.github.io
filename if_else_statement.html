if.... else statement


類似中文的如果。如果括弧裡的判斷是是對的，則進入之後的程式碼。不然則跳過並且執行接下來的程式碼。
if的語法：
if(判斷式){
  .....//code
}
....//code after if statement

if的使用方法：

ex:

int test;
test = 10;
if(test<15){
  printf("test 比15小\n");
}
printf("if statement is easy\n");
output:

再來是搭配else的用法，類似中文，如果....，不然....。


if搭配else的語法：
if(判斷式){
  .....//code
}
else{
....//code
}
....//code after if_else statement

if_else的使用方法：

ex:

int test;
test = 10;
if(test>15){
  printf("test 比15大\n");
}
else{
  printf("test 沒有比15大\n");
}
printf("if_else statement is easy\n");
