基本資料型態


c語言裡基本的資料型態有以下三種：

inteiger(整數，ex:43,5）
floatint-point numbers(浮點數，簡單一點就是含有小數，ex:3.88）
characters(字元，ex:"a", "b", "D", "5', .....)

特別注意的是數字可以是字元也可以是整數。ex:5,"5"。

宣告方法：
int a = 50;
float b = 30.45;
char c = "r";

練習一下：

請輸入下列程式碼：（不要用複製貼上，一定要親自打一遍喔～）

#include<stdio.h>

int main()

{
  int a;
  float b;
  char c;
  printf("請輸入整數a:\n");
  scanf("%d",&a);
  printf("請輸入浮點數b:\n");
  scanf("%f",&b);
  printf("請輸入字元c:\n");
  scanf(" %c",&c);
  printf("%d %f %c\n", a, b, c);
  return 0;
}
\n代表會換行。
每輸入一個值就要按一次ener。（但是字元的部分會讀到enter鍵，所以要在%c前面加一個空格，才不會讀到上一個浮點數的enter鍵）
確認輸出的a,b,c跟輸入的有沒有一樣。
