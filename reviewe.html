#include <stdio.h>

int main()

{

  int a;

  scanf("%d",&a);

  printf(a);

  return 0;

}

首先我們可以看到第一行的#include <stdio.h>，這行的作用是把stdio.h函式庫裡的函式全部包到這個檔案裡，因為我們待會會用到裡面的函式。

第二行為int main()，main()是一個function代表程式會從這個function開始執行，前面的int代表main() function會回傳一個int型態的數值。

第三行的int a為宣告一個型態為int的變數，在我們使用任和變數之前都一定要宣告，不然電腦會不知道為宣告的變數是什麼東東。

第四行的scanf("%d",&a)代表我們用鍵盤輸入一個數值，例如25，此時就會把這個數值傳給a，也就是，a = 25，注意&的符號，他代表把輸入的值傳給a的地址，每一個變數都有自己在電腦裡儲存位置的地址，當我們把25放到這個a的地址時，a的直就會是25。

第五行的printf(a)代表把a的數值給輸出出來，所以如果剛剛輸入25，此時就會輸出25。

要注意的是scanf()函式與printf()函式都屬於stdio.h函式庫裡的函式，所以一開始已定要include stdio.h才能使用
