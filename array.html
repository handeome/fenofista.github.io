陣列(array)


基本的陣列(array)分為三種：整數陣列，浮點數陣列，字元陣列。
我們可以把陣列想成是一輛火車，每節車廂都可以載一個元素，假設有10節車廂，則能載十個元素。
整數陣列：火車的每節車廂只能載整數。
浮點數陣列：火車的每節車廂只能載浮點數。
字元陣列：火車的每節車廂只能載字元。
來看看幾個例子吧！
首先要如何宣告一個陣列呢？
一維陣列的宣告如下:（之後會說明二維陣列，三維陣列.....)
  int array_int[10];（代表可以裝10個元素的整數陣列）
  float array_float[10];（代表可以裝10個元素的浮點數陣列）
  char array_char[10];（代表可以裝10個元素的字元陣列）
再來要如何使用陣列呢？
我們可以這樣做：
  array_int[3] = 7;(代表把整7數裝進陣列array_int的第三個位置（第三節車廂)。
要注意的是，陣列裡的位置是從0開始的，也就是說如果array_int只能裝10個元素的話，只有:array_int[0], array_int[1]......, array_int[9]。 沒有array_int[10]喔！！
再來我們把陣列裡每個位置都裝一個整數吧。
array_int[0] = 10;
array_int[1] = 9;
array_int[2] = 8;
array_int[4] = 6;
array_int[5] = 5;
array_int[6] = 4;
array_int[7] = 3;
array_int[8] = 2;
array_int[9] = 1;
接下來把陣列裡的元素都輸出看看，會是什麼樣子呢？
這邊就舉例輸出array_int[3]：
  printf("array_int[3] = %d",array_int[3]);
